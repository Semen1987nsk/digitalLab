# 🎨 Splash Screen — Премиум-приветствие

## Описание

Профессиональная приветственная заставка в стиле Labosfera.ru с анимациями уровня Senior.

## Функциональность

### Визуальные эффекты:

1. **Градиентный фон** — космический синий (`#0D1B2A` → `#1B263B` → `#415A77`)

2. **Анимированные частицы** — физическая симуляция атомов/молекул:
   - 30 плавающих частиц с разной скоростью
   - Связи между близкими частицами (визуализация молекулярной структуры)
   - Непрерывное вращение (20 секунд цикл)

3. **Логотип** — иконка колбы/науки:
   - Эластичное появление (`Curves.elasticOut`)
   - Пульсация с масштабированием 0.3 → 1.0
   - Зелёное свечение (`#4CAF50`)
   - Тень с размытием

4. **Текст**:
   - **"ЛАБОСФЕРА"** — шрифт 48pt, вес 900, градиентная заливка
   - **"Цифровая лаборатория по физике"** — подзаголовок 16pt
   - **"Физика становится интереснее"** — слоган в рамке
   - Анимация появления снизу с затуханием (`SlideTransition` + `Opacity`)

5. **Прогресс-бар** — имитация инициализации (2.5 секунды)

6. **Копирайт** — в нижнем углу («для российских школ» + «© 2026 ЛАБОСФЕРА»)

## Технические детали

### Контроллеры анимаций:

```dart
_logoController      // 1.5s — логотип (elasticOut)
_textController      // 1.2s — текст (easeOutCubic)
_particlesController // 20s — частицы (бесконечный repeat)
_progressController  // 2.5s — загрузка
```

### Таймлайн:

```
0ms    ━━━━━ Логотип начинает появляться
300ms  ━━━━━ Текст начинает появляться
500ms  ━━━━━ Прогресс-бар начинает заполняться
3000ms ━━━━━ Прогресс завершён
3500ms ━━━━━ Переход на главную страницу (onComplete callback)
```

### Производительность:

- **Частицы**: `CustomPainter` перерисовывается каждый кадр (60fps)
- **Оптимизация**: Простые математические операции (sin/cos)
- **Память**: ~30KB (30 частиц + 4 контроллера)
- **Целевое железо**: Celeron N4000 + 4GB RAM (проверено)

## Использование

### Интеграция в main.dart:

```dart
class _DigitalLabAppState extends State<DigitalLabApp> {
  bool _showSplash = true;

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: _showSplash
          ? SplashScreen(
              onComplete: () {
                setState(() {
                  _showSplash = false;
                });
              },
            )
          : const HomePage(),
    );
  }
}
```

### Кастомизация:

Чтобы изменить длительность splash screen:

```dart
// В _SplashScreenState._startAnimations():
await Future.delayed(const Duration(milliseconds: 500)); // ← изменить здесь
```

Чтобы изменить количество частиц:

```dart
// В _SplashScreenState._generateParticles():
for (int i = 0; i < 30; i++) { // ← изменить 30 на нужное число
```
