# 🔧 HARDWARE — Рекомендации по оборудованию

## 🎯 ПРИНЦИПЫ ПРОЕКТИРОВАНИЯ

### Ключевые требования
1. **Модульность** — каждый датчик — отдельное устройство
2. **Надёжность** — защита от КЗ, переполюсовки, перенапряжения
3. **Доступность** — компоненты легко купить
4. **Ремонтопригодность** — легко заменить сломанные части
5. **Масштабируемость** — от 1 до 100+ датчиков

### Стратегия разработки
**MVP:** Arduino + готовые модули  
**v2.0:** Собственные PCB  
**v3.0:** Интегрированные решения (всё в одном корпусе)

## 🔌 ДАТЧИКИ — Подробные спецификации

### 1. Датчик расстояния

#### Вариант A: VL53L0X (ToF лазер) — ⭐ РЕКОМЕНДУЕТСЯ
- **Диапазон:** 0-200 см
- **Точность:** ±3%
- **Интерфейс:** I2C
- **Цена:** $3-5
- **Плюсы:** Точный, не зависит от цвета/текстуры
- **Минусы:** Дороже ультразвука

```cpp
#include <Wire.h>
#include <VL53L0X.h>

VL53L0X sensor;

void setup() {
  Wire.begin();
  sensor.init();
  sensor.setTimeout(500);
  sensor.startContinuous();
}

void loop() {
  uint16_t distance = sensor.readRangeContinuousMillimeters();
  // distance в мм
}
```

#### Вариант B: HC-SR04 (Ультразвук) — БЮДЖЕТНЫЙ
- **Диапазон:** 2-400 см
- **Точность:** ±0.3 см
- **Интерфейс:** Trigger/Echo
- **Цена:** $1-2
- **Плюсы:** Дешёвый, простой
- **Минусы:** Зависит от препятствия, медленный

```cpp
const int trigPin = 9;
const int echoPin = 10;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}

float getDistance() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  long duration = pulseIn(echoPin, HIGH);
  float distance = duration * 0.034 / 2; // см
  return distance;
}
```

### 2. Датчик температуры

#### Вариант A: DS18B20 (1-Wire) — ⭐ РЕКОМЕНДУЕТСЯ
- **Диапазон:** -55°C до +125°C
- **Точность:** ±0.5°C
- **Интерфейс:** 1-Wire
- **Цена:** $2-3
- **Плюсы:** Цифровой, точный, водонепроницаемый
- **Минусы:** Медленный (750 мс на измерение)

```cpp
#include <OneWire.h>
#include <DallasTemperature.h>

OneWire oneWire(2);
DallasTemperature sensors(&oneWire);

void setup() {
  sensors.begin();
}

float getTemperature() {
  sensors.requestTemperatures();
  return sensors.getTempCByIndex(0);
}
```

#### Вариант B: BME280 (температура + влажность + давление)
- **Диапазон:** -40°C до +85°C
- **Точность:** ±1°C, ±3% RH, ±1 hPa
- **Интерфейс:** I2C / SPI
- **Цена:** $5-8
- **Плюсы:** 3 в 1, высокая точность
- **Минусы:** Дороже

### 3. Датчик света

#### BH1750 (Ambient Light) — ⭐ РЕКОМЕНДУЕТСЯ
- **Диапазон:** 1-65535 lux
- **Интерфейс:** I2C
- **Цена:** $2-3
- **Плюсы:** Точный, широкий диапазон, цифровой

### 4. Датчик давления

#### BMP280 — ⭐ РЕКОМЕНДУЕТСЯ
- **Диапазон:** 300-1100 hPa
- **Точность:** ±0.12 hPa (±1 м высота)
- **Интерфейс:** I2C / SPI
- **Цена:** $3-5
- **Плюсы:** Точный, есть температура
- **Минусы:** —

```cpp
#include <Wire.h>
#include <Adafruit_BMP280.h>

Adafruit_BMP280 bmp;

void setup() {
  bmp.begin(0x76);
}

float getPressure() {
  return bmp.readPressure() / 100.0; // hPa
}

float getAltitude() {
  return bmp.readAltitude(1013.25); // м
}
```

### 5. Микрофон (Звук)

#### MAX4466 (Аналоговый) — ⭐ РЕКОМЕНДУЕТСЯ
- **Диапазон:** 20 Hz - 20 kHz
- **Интерфейс:** Аналоговый (АЦП)
- **Цена:** $2-3
- **Плюсы:** Простой, качественный звук
- **Минусы:** Требует калибровки для дБ

```cpp
const int micPin = A0;

void setup() {
  pinMode(micPin, INPUT);
}

float getSoundLevel() {
  int sampleWindow = 50; // мс
  int sample;
  unsigned long startMillis = millis();
  unsigned int peakToPeak = 0;
  unsigned int signalMax = 0;
  unsigned int signalMin = 1024;
  
  // Собираем пики за период
  while (millis() - startMillis < sampleWindow) {
    sample = analogRead(micPin);
    if (sample > signalMax) signalMax = sample;
    if (sample < signalMin) signalMin = sample;
  }
  
  peakToPeak = signalMax - signalMin;
  
  // Конвертация в дБ (приблизительно)
  float volts = (peakToPeak * 5.0) / 1024.0;
  float dB = 20 * log10(volts / 0.00631);
  
  return dB;
}
```

### 6. Акселерометр/Гироскоп

#### MPU6050 (6-осевой) — ⭐ РЕКОМЕНДУЕТСЯ
- **Акселерометр:** ±2/4/8/16 g
- **Гироскоп:** ±250/500/1000/2000 °/s
- **Интерфейс:** I2C
- **Цена:** $2-4
- **Плюсы:** Дешёвый, популярный, много библиотек
- **Минусы:** Требует калибровки

```cpp
#include <Wire.h>
#include <MPU6050.h>

MPU6050 mpu;

void setup() {
  Wire.begin();
  mpu.initialize();
}

void getAcceleration(float &x, float &y, float &z) {
  int16_t ax, ay, az;
  mpu.getAcceleration(&ax, &ay, &az);
  
  x = ax / 16384.0; // g
  y = ay / 16384.0;
  z = az / 16384.0;
}
```

## 🔋 ПИТАНИЕ

### Варианты питания

#### 1. USB (5V) — ⭐ РЕКОМЕНДУЕТСЯ для MVP
- **Источник:** Компьютер через USB
- **Плюсы:** Просто, не нужны батареи
- **Минусы:** Привязан к ПК

#### 2. Батарейки (AA/AAA)
- **Напряжение:** 3V (2xAA) или 4.5V (3xAA)
- **Плюсы:** Автономность
- **Минусы:** Нужен регулятор напряжения

#### 3. Li-Ion аккумулятор (18650)
- **Напряжение:** 3.7V
- **Ёмкость:** 2000-3500 mAh
- **Плюсы:** Перезаряжаемый, долгий
- **Минусы:** Нужна защита от переразряда

### Рекомендуемая схема питания (автономная)

```
Li-Ion 18650 (3.7V)
    ↓
TP4056 (зарядка)
    ↓
Защита от переразряда
    ↓
Boost converter (→ 5V для Arduino)
    ↓
LDO (→ 3.3V для датчиков)
```

**Время работы:**  
3000 mAh / 100 mA = 30 часов

## 🧪 КАЛИБРОВКА ДАТЧИКОВ

### Датчик расстояния (VL53L0X)
1. Измерить известное расстояние (50 см)
2. Получить значение с датчика
3. Рассчитать offset: `offset = measured - reference`
4. Применять: `real_distance = measured - offset`

### Термометр (DS18B20)
1. Поместить в тающий лёд (0°C)
2. Поместить в кипящую воду (100°C)
3. Линейная калибровка: `T_real = a * T_measured + b`
